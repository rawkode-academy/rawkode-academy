Iroh is a peer-to-peer networking stack for connecting devices directly across local networks, the public internet, and relay paths when direct connectivity is not available.

The project provides end-to-end encrypted QUIC connections, endpoint addressing, discovery, relays, and higher-level protocols for blobs, gossip, and realtime documents. It is aimed at applications that need direct device-to-device communication without treating IP addresses as the primary application identity.

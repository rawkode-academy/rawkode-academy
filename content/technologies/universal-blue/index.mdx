Universal Blue is a family of custom Fedora Atomic desktop images built with cloud-native tooling. Instead of installing Fedora Silverblue or Kinoite and then tweaking it by hand, you rebase to an OCI image maintained on GitHub, and ostree pulls the next version atomically on reboot.

The images are assembled in GitHub Actions by layering RPMs, Flatpaks, drivers, and configuration on top of the Fedora base, then pushed to a container registry as signed OCI artifacts. Because the whole OS is an immutable container image, rollbacks are trivial (pin to the previous tag) and drift between machines effectively disappears. The project maintains several curated variants: Bluefin (GNOME, developer workstation), Aurora (KDE Plasma), Bazzite (gaming, Steam Deck-style with HDR and controller support), and uCore (a server variant based on Fedora CoreOS).

Universal Blue is a practical demonstration that the container build-and-ship model works for desktop Linux, not just for server workloads.
